1. sequential access by APIs:

step1:
  • List first page(10 items per page) http://localhost:8080/api/books?page=0&size=10
  • List second page(2 items per page)http://localhost:8080/api/books?page=1&size=2
step2:
  • Login as Rambo(pls save Token)http://localhost:8080/api/accounts/login?email=rambo.zhu@erian.io&password=springside
  • Rambo request to borrow David's book (replace the Token in URL as previous saved value)http://localhost:8080/api/books/3/request?token=replaceme
  • List all books borrowed by Cavlin(replace the token)http://localhost:8080/api/myborrowedbook?token=replaceme
step3:
  • Login as Davidhttp://localhost:8080/api/accounts/login?email=david.wang@erian.io&password=springside
  • Confirm borrow by Davidhttp://localhost:8080/api/books/3/confirm?token=replaceme
  • List all books borrowed by Davidhttp://localhost:8080/api/mybook?token=replaceme
step4:
  • Rambo return the book so others can borrowhttp://localhost:8080/api/books/3/return?token=replaceme
  • List the current status of the bookhttp://localhost:8080/api/books/3
other flow:
  • Rambo cancel the apply, replace the tokenhttp://localhost:8080/api/books/3/cancel?token=replaceme
  • David reject the apply, replace the tokenhttp://localhost:8080/api/books/3/reject?token=replaceme
